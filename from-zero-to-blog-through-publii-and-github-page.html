使用 Publii + Github Page 從零開始自架部落格

沒有技術背景也能輕鬆架站！用 Publii + GitHub Pages 免費打造你的個人部落格 🚀

你是不是也曾經想過：「我也想寫部落格，但我又不會寫程式，也不會搞那些主機、DNS 的東西啊……」

別擔心！今天要介紹的這個工具組合，不但完全免費，而且零技術背景也能輕鬆上手——只要會滑鼠拖拉，就能完成架設你的第一個部落格！

我們要用到的工具是：

🔧 Publii：一款免費的離線部落格製作工具
☁️ GitHub Pages：免費的網站空間，只要有帳號就能用

一、什麼是 Publii？為什麼推薦它？

Publii 是一套免費桌面應用程式（支援 Windows / macOS / Linux），可以幫你建立一個現代化、有質感的靜態網站。你不用寫任何一行程式碼，像寫 Word 一樣輸入文章內容，再點幾下滑鼠就能發佈。

特色：

✅ 完全免費
✅ 支援離線編輯，不用開網頁就能寫文章
✅ 超多漂亮主題可以直接套用
✅ 一鍵發佈到 GitHub Pages、Netlify、S3 等平台

二、準備工作（只要做一次）

步驟 1：安裝 Publii

  1. 前往官方網站：https://getpublii.com

  3. 安裝並打開 Publii

步驟 2：建立 GitHub 帳號 & Repository

  1. 前往 https://github.com 註冊帳號

  2. 登入後，點選右上角 ➕ > New repository

  3. Repository 名稱請取成：你的帳號.github.io（這是 GitHub Pages 的規則）

  4. 其它選項可以保持預設，點「Create repository」

三、開始用 Publii 建立你的部落格

步驟 1：建立新網站

  1. 開啟 Publii，點擊「Create a new website」

  2. 輸入網站名稱

  3. 建置完成應該可以看到這個畫面

步驟 2：撰寫你的第一篇文章 ✍️

  1. 點選「Posts」>「New Post」

  2. 輸入標題、內容、封面圖片等資訊

  3. 記得按下「Save」(你也可以選擇存成草稿稍後再發佈)

步驟 3：設定 GitHub Pages 發佈

  1. 點選左側「Server」>「Github Pages」

  2. 輸入你的 GitHub 帳號、Repository 名稱、Email、Token（第一次需要建立）

 

📌 你可以在這裡建立 Token：https://github.com/settings/tokens

  • 給這個 Token 起個名字並選擇 repo 權限

步驟 4：一鍵發佈！

  1. 可以先按「Preview your changes」來預覽目前的部落格

  2. 確認沒問題後，點擊「Sync your website」

  3. 第一次會稍久，之後都會更快

  4. 完成後，你就可以用網址 https://[你的帳號].github.io 開啟網站啦！

🎉 恭喜你，你的個人網站完成了！

 

現在你已經學會怎麼：

  • 建立一個漂亮的網站

  • 撰寫與管理文章

  • 發佈到 GitHub Pages，讓全世界都能看到你的創作

最重要的是——這整個過程你不用寫任何程式碼！
